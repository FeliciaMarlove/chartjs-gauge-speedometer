### length

• **length**: `string` \| `number`

Needle length.
String ending with '%' means percentage of the chart radius, number means pixels.

**`Default`**

'80%'

___

### radius

• **radius**: `string` \| `number`

Needle circle.
String ending with '%' means percentage of the chart radius, number means pixels.

**`Default`**

'10%'

___

### width
